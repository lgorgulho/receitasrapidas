File Flambado ao Molho de Wiski e Cereja

Receitas Rápidas February 15, 2009 Receitas Carnes

Ingredientes

Ingredientes.1 file mignon inteiro 1.800
mostarda amarela.sal temperado
oleo de soja.1 copo de whiski
2 vidros de cerejas ao marasquino de 250g (140g drenado)
panela de ferro com tampa.

Modo de Preparo

Tempere o file cobrindo uma de suas faces com o sal temperado a
outra face, cubra com uma fina camada de mostarda amarela. Deixe
descansar por quatro horas.
Coloque a panela de ferro em fogo medio, cobrindo seu fundo com
óleo. Quando o oleo estiver aquecido, coloque o filé e tampe. A
cada 5 minutos, vire o file para que nao queime. Se necessario,
adicione mais oleo.
Quando o filé mostrar-se firme ao toque do garfo, ele estara
cozido. Este ponto deve ocorrer entre 40 a 45 minutos.
Quando o file estiver cozido,retire da panela para fatiar
adicione na panela um pouco dagua formando um caldo
reserve.Volte o file com azeite para fritar a parte que nao havia
tido contato com a panela  adicione a calda da cereja e as
cerejas de um copo reserve a cereja do outro.Flambe com o wiski
Quando o caldo começar a ferver, coloque um fosforo aceso no
vapor que se desprende queimando o alcool existente no molho. Com
um garfo comprido vire o file para que o álcool existente no
whisky queime completamente. Quando nao houver mais chama o prato
estara pronto junte entao um pouco.do primeiro caldo reservado e
as cerejas não flambadas
Em uma travessa sirva com arroz, salada e batata palha.
Servir somente com batata palha e bastante molho sobre ela também.

  • Comida: Brasileira
  • Tempo de Preparo: 45 min
  • Tipo de Preparo: Assados
  • Preço: R$60,00
